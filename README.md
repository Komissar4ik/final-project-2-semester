# Выпускной проект

Проверьте, что ваш проект соответствует ожидаемым требованиям (подробнее в [презентации](kursovie-t-academy-front.pptx)).

Укажите всю необходимую информацию о проекте, которая потребуется на защите в RADME.md:
- ФИО создателя проекта
- Указание, что проект является курсовой работой на курсе "Т-Академия Фронтенда"
- Название веб-приложения
- Ссылка на общедоступный ресурс с готовым приложением
- Задача, которую оно призвано решить
- Описание целевой аудитории приложения
- Краткое описание возможностей приложения
- Инструкции по запуску проекта
- Информация с тестовыми данными для использования приложения
- Другая информация на ваше усмотрение
